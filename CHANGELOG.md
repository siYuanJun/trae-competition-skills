# 更新日志

## [1.1.0] - 2026-06-18

### 🚀 评分优化升级
- **新增 `scripts/scoring-optimizer.md`**：四维度评分预检工具（创新性/实用性/完成度/表现力），定位弱项并给出定向优化建议
- **新增 `references/scoring-guide.md`**：评分标准详细解读 + 各维度高分策略 + 高频低分陷阱提醒
- **新增 `references/design-guide.md`**：HTML 页面设计规范（色彩/字体/间距/动画/无障碍），按赛道差异化设计

### 🔧 工具升级
- `select-topics.md`：新增「创新性约束」章节 + 创新性评级输出，拒绝换皮型方案
- `generate-post.md`：新增用户使用故事、有无对比、数据佐证等实用性增强要求
- `generate-html.md`：集成 design-guide，细化到色彩/字体/间距/交互反馈级别
- `quality-check.md`：从「合规检查」升级为「合规+完成度+创新性复核」三轮评审
- `SKILL.md`：新增第5阶段「评分预检与高分优化」，核心铁律加入双目标驱动

### 📊 流程变化
- 原有5阶段 → 升级为6阶段（新增阶段5：评分预检）
- 产出新增：四维度评分报告、优化前后对比表、创新性评级
