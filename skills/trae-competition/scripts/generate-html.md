# TRAE Work Auto模式 HTML生成提示词

页面定位：根据选题类型定位为轻量化科普/展示/工具单页，适配手机+电脑双端响应式

## 设计规范（必须遵守）
参考 `references/design-guide.md` 中的详细规范，核心要求：

### 色彩
- 主色+辅色不超过3种，灰度系统布局
- 按赛道应用差异化配色（社会服务/公益=暖色、学习工作=冷色、生活娱乐=活泼、硬件=科技暗色）

### 字体
- 标题 24-32px / 副标题 18-20px / 正文 15-16px（适老类≥18px）
- 行高 ≥1.6，确保阅读舒适
- 字体栈：`-apple-system, "PingFang SC", "Microsoft YaHei", sans-serif`

### 间距
- 页面内边距 16-20px（手机）/ 24-32px（PC）
- 卡片间距 16-20px，段落间距 16-24px
- 按钮内边距 ≥10px 垂直 / 24px 水平

### 交互
- 按钮点击有 scale(0.97) 反馈
- 输入框聚焦有边框变色过渡（0.2s ease）
- 结果展示有淡入动画（opacity 0→1, 0.3s ease）
- 移动端触控区域 ≥44×44px

### 无障碍
- 所有图片含 alt 属性
- 颜色不是唯一信息区分方式
- 适老场景对比度 ≥4.5:1

### 代码质量
- 单文件 HTML+内置 CSS/JS，无外部依赖
- CSS 变量管理颜色/间距/圆角
- 中文注释清晰，按区块组织
- 移动端优先，≥768px / ≥1024px 断点适配

### 禁止事项
- 外部 CDN 依赖（通用图标库除外，须注明）
- 内联样式散落（统一放 `<style>` 块）
- 花哨动画（单个动效 ≤0.3s）
