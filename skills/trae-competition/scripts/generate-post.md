# 赛事发帖文案生成提示词

前置规则：大赛官方发帖模板、审核规则、支持全赛道【学习工作、生活娱乐、社会服务、社会公益、硬件交互】

输入素材：{替换为单个创意选题信息，含创新性亮点}

## 强制输出内容

### 1. 合规标题
【对应赛道名称】+ 创意名称

### 2. 标签勾选说明
基础赛道选对应赛道，附加社会公益标签（如适用）

### 3. 完整正文（≥120字，严格4大模块，每个模块内部细分小点全部写满）

**模块1：创意名称 + 介绍**
- 1-2句话描述要解决的具体痛点（数据佐证更佳）
- 说明创意灵感、创作动机，以及**为什么这个角度别人没想到**（创新性）
- 一句话概括产品形态（网页交互工具/可视化科普页，适配TRAE Work导出HTML）

**模块2：目标用户 + 痛点**
- 精准描述核心受益用户群体（年龄、身份、特征）
- **用户使用故事**：用一个具体人物的一天串联使用流程（如「张阿姨早上打开页面→输入症状→...」）
- **有无对比**：无本产品时用户的困境 vs 有本产品后的改善

**模块3：价值与意义**
- 社会价值角度：受益人群规模、贴合政策方向
- 实用价值角度：效率提升、成本降低、体验改善
- 如有公开数据可引用作为论据支撑

**模块4：TRAE Work HTML 物料说明**
- Auto模式生成流程简述
- 页面核心交互功能（按赛道指定设计要求）
- 跨设备适配说明
