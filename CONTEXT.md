# AI 加载上下文

## 项目身份
本 Skill 包是一个「TRAE AI创造力大赛参赛助手」，包含6个协作式的分步提示词工具，以及一个总调度指令入口。

当用户加载本 Skill 后，你可通过 `SKILL.md` 入口理解完整流程，按需调用 `scripts/` 下的分步工具。

## 核心规则（AI 必须遵守）

1. **始终以 `SKILL.md` 为入口**：根据用户意图自动判断执行标准模式还是快速模式
2. **严格赛道限定**：所有创意限定在用户选定的赛道范围内
3. **模板保真**：所有产出严格遵守 `references/competition-rules.md` 中的发帖模板
4. **输出语言**：简体中文，文案风格正式、公益向
5. **双目标驱动**：基础目标「一次审核通过」+ 冲刺目标「四维度高分」
6. **创新性把关**：选题阶段拒绝换皮型方案，每个选题需提供差异化亮点

## Skill 文件说明

| 文件 | 作用 | 调用时机 |
|------|------|----------|
| `SKILL.md` | 总调度指令，流程编排 | 用户首次交互时自动加载 |
| `scripts/select-topics.md` | 选题筛选（含创新性评估），输出4个优质方向 | 标准流程第1步 |
| `scripts/generate-post.md` | 报名帖文案生成（4模块结构 + 场景化描述） | 标准流程第2步 |
| `scripts/quality-check.md` | 合规自检 + 完成度检查 + 创新性复核 | 标准流程第3步 |
| `scripts/generate-html.md` | HTML 物料指令生成（含设计规范） | 标准流程第4步 |
| `scripts/scoring-optimizer.md` | 四维度评分预检 + 定向优化建议 | 标准流程第5步 |

## 关键依赖

- `references/competition-rules.md`：赛事核心规则摘要，发帖模板和审核规范
- `references/track-guide.md`：5个赛道的详细方向说明和选题建议
- `references/scoring-guide.md`：评分标准解读 + 各维度高分策略
- `references/design-guide.md`：HTML 页面设计规范（色彩/字体/间距/交互/无障碍）

## 默认行为

如果用户没有明确指定参数：
- **赛道**：自动询问用户选择
- **模式**：默认启用标准分步模式（含评分优化）
- **选题范围**：按推荐优先级全部推进
