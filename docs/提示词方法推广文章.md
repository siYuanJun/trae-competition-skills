# 我把 TRAE 比赛参赛流程做成了一个开源 Skill 包，拿去用

如果你正在参加 TRAE AI 创造力大赛，大概率经历过这种循环：打开选题界面 → 脑子一片空白 → 关掉 → 第二天重复。

我索性把整个参赛流程做成了一个结构化的 AI Skill 工程包，开放在 GitHub 上。从选题、写文案、质检到生成 HTML 物料，全部交给 AI 自动跑。零代码、开箱即用、审核一次过。

---

## 这个项目是什么

**`trae-competition-skills`** 是一套遵循 Skill 工程标准的 AI 提示词工具包。它不是零散的几段提示词，而是一个有目录结构、有元数据标注、有校验脚本的正式项目。

```bash
# 三行命令即可用
git clone https://github.com/siYuanJun/trae-competition-skills.git
# 在 Trae / Claude Code 中加载 skills/trae-competition/ 目录
# 告诉 AI：用标准模式，赛道选社会服务
```

AI 会自动按**选题 → 文案 → 质检 → 物料 → 汇总**五个阶段一步步产出，每一步都可以中途调整。如果你赶时间，一句「快速出稿」就能跳过中间确认环节，一次性拿到全套材料。

## 为什么做成 Skill 工程，而不是几段提示词

散装的提示词最大的问题是**没法复用**——文件名混乱、内容耦合、换个 AI 工具就没法用。

这个项目参考了 `mattpocock/skills` 等主流开源 Skill 仓库的目录规范，做了四件事：

### 1. 标准化目录结构
```
trae-competition-skills/
├── skills/trae-competition/
│   ├── SKILL.md                  # ★ 主入口：总调度指令
│   ├── scripts/
│   │   ├── select-topics.md      # 阶段1：选题筛选
│   │   ├── generate-post.md      # 阶段2：文案生成
│   │   ├── generate-html.md      # 阶段4：HTML物料
│   │   └── quality-check.md      # 阶段3：合规质检
│   └── references/
│       ├── competition-rules.md  # 赛事规则参考
│       └── track-guide.md        # 赛道详细说明
├── docs/                         # 中文文档
├── scripts/                      # 工具脚本
├── CONTEXT.md                    # AI 加载说明
└── README.md                     # 你正在看的这个
```

每个文件带 YAML 元数据头，AI 工具可以精确理解它的用途和调用时机。

### 2. 质量保障内置
专门的质检工具（`quality-check.md`）会对生成的每篇帖子逐项检查：4大模块是否完整、字数是否达标、有无敏感表述——等于把审核官的标准写进了提示词里，不用自己对着规则一条条核对。

### 3. 带校验 + 打包脚本
- `npm run validate` → 一键检查项目文件完整性
- `npm run pack` → 自动打包为 `.tar.gz` 发布包
- GitHub Actions CI → 每次提交自动运行校验

### 4. 赛事规则独立维护
所有官方审核规则、发帖模板、赛道说明都抽离到 `references/` 目录。万一赛事规则更新，改一份参考文件就行，核心逻辑不用动。

## 怎么上手

### 第一步：克隆项目
```bash
git clone https://github.com/siYuanJun/trae-competition-skills.git
cd trae-competition-skills
```

### 第二步：加载 Skill
- **Trae**：Skill 面板 → 导入 → 选择 `skills/trae-competition/` 目录
- **Claude Code**：`claude --load skills/trae-competition/SKILL.md`
- **其他 AI 工具**：把 `SKILL.md` 的内容粘贴为系统提示词

### 第三步：开始干活
```
用标准模式，赛道选社会服务
```

AI 会引导你走完全流程。想偷懒？直接说「快速出稿，赛道选社会公益」，一次拿全部材料。
