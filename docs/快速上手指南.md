# 快速上手指南

> 📖 3步搞定：从下载到产出第一份参赛材料。

---

## 第1步：获取项目

### 方式A：直接克隆（推荐）

```bash
git clone https://github.com/你的用户名/trae-competition-skills.git
cd trae-competition-skills
```

### 方式B：下载压缩包

从 [GitHub Releases](https://github.com/你的用户名/trae-competition-skills/releases) 页面下载最新版 `.tar.gz`，解压备用。

---

## 第2步：加载 Skill 到你的 AI 助手

### 在 Trae 中使用

1. 打开 Trae，进入 Skill 管理面板
2. 点击「导入 Skill」，选择 `skills/trae-competition/` 目录
3. Skill 自动注册，名称显示为 `trae-competition`

### 在 Claude Code 中使用

```bash
claude --load skills/trae-competition/SKILL.md
```

### 在其他 AI 助手中使用

将 `skills/trae-competition/SKILL.md` 的文件内容作为系统提示词或上下文粘贴到对话中。

---

## 第3步：告诉 AI 你的需求

### 新手推荐 → 标准分步模式

在 AI 对话中输入：

```
用标准模式，赛道选社会服务
```

AI 会依次执行：
1. **选题定向** → 产出4个优质选题供你参考
2. **文案生成** → 针对确认的选题生成报名帖
3. **合规质检** → 逐项检查，修正格式问题
4. **物料生成** → 输出 TRAE Work HTML 指令
5. **交付汇总** → 汇总全部成果清单

每一步你都可以中途调整方向。

### 老手快捷 → 快速一键模式

```
快速出稿，赛道选社会公益
```

AI 一次性输出全套材料，适合对流程已经熟悉的用户。

## 常见问题

### Q: 我不确定选什么赛道？
A: 直接说「帮我推荐一个赛道」，AI 会根据2026年政策方向给你建议。新人推荐从「社会服务」开始。

### Q: 产出的帖子能直接用吗？
A: 可以。所有文案已严格按官方模板生成，但建议你通读一遍，根据自己的想法微调创意细节。

### Q: HTML 文件怎么导出？
A: AI 生成的指令可直接粘贴到 TRAE Work 中执行，自动产出 `.html` 文件，右键即可下载。

### Q: 能用这套工具参加多个赛道吗？
A: 可以。换个赛道参数重新跑一次即可，每个赛道产出4个选题。

### Q: 提示词生成的内容会不会重复？
A: 每次运行 AI 都会基于最新上下文生成新内容，不会和上一次完全相同。
